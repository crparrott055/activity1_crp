## Step 6
Why Web Design Matters

A well-designed website considers its target audience, purpose, and structure. Planning is a critical step in the design process and includes creating a wireframe and a site map to organize content effectively

Effective websites use consistent navigation, clear typography, and balanced color schemes to improve readability and usability. These elements help guide users through the site and ensure they can find information quickly.

Understanding the Web

The World Wide Web consists of interconnected webpages accessed through web browsers. Each webpage is identified by a URL and delivered using protocols such as HTTP and TCP/IP, which allow communication between servers and clients.

Web browsers interpret HTML code and display it visually, allowing users to interact with text, images, and links.
